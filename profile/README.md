## [winkNLP](https://github.com/winkjs/wink-nlp)

#### *Developer-friendly Natural Language Processing.*

A high-performance NLP pipeline — tokenization, sentence boundary detection, negation handling, sentiment analysis, POS tagging, NER, and custom entities — in a single zero-dependency package with full TypeScript support. The entire pipeline runs at **1M tokens per second**, identically in Node.js and browsers. [Run the live benchmark](https://observablehq.com/@winkjs/how-to-measure-winknlps-speed-on-browsers) on Observable. Explore the [documentation](https://nlp.winkjs.org) and [recipes](https://observablehq.com/collection/@winkjs/winknlp-recipes).

## [winkComposer](https://github.com/winkjs/composer)

#### *Composable Streaming Intelligence.*

From IIoT streams to actionable insight — in real time. Composable blocks for signal conditioning, statistics, anomaly detection, and flow control — from Kalman filters to spiking neural networks. Express *what* you want, not *how* to build it; AI-native via MCP Server. A change point detection pipeline runs at **1M messages per second on a modern server**, **100K on a Raspberry Pi** — same code, edge to cloud. [Run the live benchmark](https://composer.winkjs.org/docs/benchmark) in your browser. Explore the [documentation](https://composer.winkjs.org) and [use cases](https://composer.winkjs.org/docs/use-cases).
